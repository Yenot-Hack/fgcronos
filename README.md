#### Installation

- Clone the repository
```
git clone https://github.com/Yenot-Hack/fgcronos.git
```
</br>

- Enter the repository
```
cd fgcronos
```
</br>

- Give execution permissions
```
chmod +x $PWD/*
```
</br>

- Run the installer
```
./install.sh
```
</br>

Usage:
to run the tool just write the command fgchronos 
```
fgcronos
```
